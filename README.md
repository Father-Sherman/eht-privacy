# eht-privacy

Public-facing pages for the **E-HT Weight Loss** Android app. This
repo exists only to host the privacy policy on GitHub Pages — the
app's source lives in a separate private repo.

## Hosted at

- Landing: <https://father-sherman.github.io/eht-privacy/>
- Privacy: <https://father-sherman.github.io/eht-privacy/privacy/>

## Editing

Edit `privacy.md` or `index.md` and push to `main`. GitHub Pages
rebuilds in ~1 minute. The "Last updated" date inside `privacy.md`
should be bumped any time the policy changes in a way users would
care about.
